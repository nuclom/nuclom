Workflow templates let you apply consistent AI processing and sharing rules to videos. Templates are currently available via the API for automation and integrations.

  </Card>

## Create a Template (API)

```http
POST /api/workflow-templates
```

```json
{
  "name": "Weekly Leadership Sync",
  "description": "Summary + action items",
  "type": "meeting_recap",
  "organizationId": "org_xxx",
  "config": {
    "autoTranscribe": true,
    "generateSummary": true,
    "extractChapters": true,
    "extractActionItems": true,
    "subtitleLanguages": ["en"]
  }
}
```

## Apply a Template (API)

```http
POST /api/workflow-templates/:id
```

<Note>
  UI support for workflow templates is on the roadmap. For now, use the API or automation tools like Zapier.
</Note>
