## Using Templates

### When Uploading

<Steps>
  <Step title="Start Upload">
    Click **New Video** in your dashboard
  </Step>
  <Step title="Select File">
    Select or drag your video file
  </Step>
  <Step title="Choose Template">
    Choose a **Workflow Template** from the dropdown
  </Step>
  <Step title="Upload">
    Template settings will be applied automatically
  </Step>
</Steps>

### Apply to Existing Video

<Steps>
  <Step title="Open Video">
    Navigate to the video you want to process
  </Step>
  <Step title="Settings">
    Click the **Settings** icon (gear)
  </Step>
  <Step title="Apply">
    Select **Apply Template** and choose your template
  </Step>
</Steps>

    | ----- | ----------- |

    | ------- | ----------- |

    | **Extract action items** | Find tasks and to-dos |

  </Tab>
  <Tab title="Sharing">
    | Setting | Description |
    | ------- | ----------- |
    | **Auto-share** | Create share link on completion |
    | **Access level** | View, comment, or download |
    | **Expiration** | Days until link expires |
  </Tab>
</Tabs>

## Template Settings Reference

### AI Processing Options

| Setting | Description | Default |
|---------|-------------|---------|
| `autoTranscribe` | Generate transcript from audio | `true` |
| `generateSummary` | Create AI-powered summary | `true` |
| `extractChapters` | Detect chapter markers | `true` |
| `extractActionItems` | Find tasks and action items | Varies |
| `detectCodeSnippets` | Identify code in video | `false` |

### Subtitle Languages

Available languages:
- English (en), Spanish (es), French (fr), German (de)
- Portuguese (pt), Japanese (ja), Chinese (zh), Korean (ko)
- Italian (it), Dutch (nl)

### Auto-Share Settings

| Setting | Options |
|---------|---------|
| `enabled` | `true` / `false` |
| `accessLevel` | `view`, `comment`, `download` |
| `expiresInDays` | Number or `null` (never) |

### Custom Prompts

Customize AI behavior with specific instructions:

**Summary Prompt Example:**

```
Generate a meeting summary that includes:
1. Key decisions made
2. Action items with owners
3. Topics discussed
4. Next steps and deadlines
```

**Action Items Prompt Example:**

```
Extract all action items, including:
- The specific task
- Who is responsible
- Due date if mentioned
- Priority if indicated
```

## API Reference

### List Templates

```http
GET /api/workflow-templates
```

Query parameters:
- `organizationId` - Filter by organization
- `type` - Filter by template type
- `system` - Include system templates (`true`/`false`)

### Create Template

```http
POST /api/workflow-templates
```

```json
{
  "name": "My Custom Template",
  "description": "For weekly team updates",
  "type": "meeting_recap",
  "icon": "Users",
  "organizationId": "org_xxx",
  "config": {
    "autoTranscribe": true,
    "generateSummary": true,
    "extractChapters": true,
    "extractActionItems": true,
    "subtitleLanguages": ["en", "es"]
  }
}
```

### Apply Template

```http
POST /api/workflow-templates/:id
```

## Troubleshooting

<AccordionGroup>
  <Accordion title="Template not applying">
    - Ensure correct organization is selected
    - Check that the template is active
    - Verify processing credits are available
  </Accordion>
  <Accordion title="Custom prompts not working">
    - Prompts should be clear and specific
    - Avoid conflicting instructions
    - Test with shorter videos first
  </Accordion>
  <Accordion title="Missing features in output">
    - Some features require longer videos
    - Code detection needs visible code
    - Action items require spoken tasks
  </Accordion>
</AccordionGroup>
