Customize your Nuclom experience with the comprehensive settings and preferences system.

## Personal Settings

### Profile Management

<Steps>
  <Step title="Navigate to Settings">
    Go to **Settings** → **Profile**
  </Step>
  <Step title="Update Profile">
    Update display name, profile picture, timezone, and contact information
  </Step>
</Steps>

**Profile Picture Options:**
- Upload custom image (JPG, PNG, GIF - max 5MB)
- Use Gravatar auto-sync
- Use initials-based avatar
- Remove picture (return to default)

### Account Security

#### Password Management

<Steps>
  <Step title="Access Security Settings">
    Go to **Settings** → **Security**
  </Step>
  <Step title="Change Password">
    Click **Change Password** and follow the prompts
  </Step>
</Steps>

**Password Requirements:**
- Minimum 8 characters
- At least one uppercase letter
- At least one lowercase letter
- At least one number
- Special characters recommended

#### Two-Factor Authentication (2FA)

<Steps>
  <Step title="Enable 2FA">
    Go to **Settings** → **Security** and click **Enable 2FA**
  </Step>
  <Step title="Scan QR Code">
    Use an authenticator app (Google Authenticator, Authy, 1Password)
  </Step>
  <Step title="Verify">
    Enter the 6-digit code to complete setup
  </Step>
  <Step title="Save Backup Codes">
    Save your backup codes securely for account recovery
  </Step>
</Steps>

<Warning>
  Backup codes are one-time use. Store them in a secure location.
</Warning>

#### Passkeys (WebAuthn)

Passkeys provide passwordless authentication using biometrics or hardware security keys.

<Steps>
  <Step title="Add Passkey">
    Go to **Settings** → **Security** and click **Add Passkey**
  </Step>
  <Step title="Name Your Passkey">
    Enter a name (e.g., "MacBook Touch ID")
  </Step>
  <Step title="Register">
    Follow your browser/device prompts to register
  </Step>
</Steps>

#### Active Sessions

View and manage all active login sessions:

| Action | Description |
| ------ | ----------- |
| **View sessions** | See device info and IP address |
| **Sign out session** | Revoke specific session |
| **Sign out all** | Secure your account by ending all other sessions |

### API and Integrations

#### API Keys

<Steps>
  <Step title="Create Key">
    Go to **Settings** → **API Keys** and click **Create API Key**
  </Step>
  <Step title="Configure">
    Enter name and select expiration (7, 30, 90 days, 1 year, or never)
  </Step>
  <Step title="Copy Key">
    Copy your API key immediately - you won't see it again!
  </Step>
</Steps>

**Using API Keys:**

```bash
curl -X GET "https://api.nuclom.com/v1/videos" \
  -H "x-api-key: nc_your_api_key_here"
```

<Note>
  - Rate limit: 100 requests/minute per key
  - All keys start with `nc_`
  - Default expiration: 30 days
</Note>

#### OAuth Applications

<Steps>
  <Step title="Create App">
    Go to **Settings** → **OAuth Apps** and click **Create Application**
  </Step>
  <Step title="Configure">
    Enter name, icon URL, and redirect URLs
  </Step>
  <Step title="Copy Credentials">
    Copy client ID and secret immediately!
  </Step>
</Steps>

**OAuth 2.0 Endpoints:**

| Endpoint | Path |
| -------- | ---- |
| **Authorization** | `/api/auth/oauth2/authorize` |
| **Token** | `/api/auth/oauth2/token` |
| **Scopes** | `openid`, `profile`, `email`, `offline_access` |

## Troubleshooting

<AccordionGroup>
  <Accordion title="Settings Not Saving">
    - Clear browser cache and cookies
    - Disable browser extensions
    - Try different browser or incognito mode
    - Check JavaScript is enabled
  </Accordion>
  <Accordion title="Missing Notifications">
    - Check notification settings
    - Verify email address
    - Check spam folders
    - Confirm preferences
  </Accordion>
  <Accordion title="Too Many Notifications">
    - Adjust notification frequency
    - Disable unnecessary notifications
    - Use digest mode
    - Set up filtering
  </Accordion>
</AccordionGroup>
