## Video Upload Process

### Supported Formats

### Video Details and Metadata

#### Required Information

| Field | Tips |
| ----- | ---- |
| **Title** | Make it descriptive and searchable. Avoid generic titles like "Video 1" |
| **Duration** | Automatically detected during upload |

#### Recommended Information

| Field | Tips |
| ----- | ---- |
| **Description** | Detailed explanation with key points and timestamps |
| **Thumbnail** | Choose from auto-generated options or upload custom (1280x720) |
| **Channel** | Organize videos by topic or project |
| **Series** | Add to sequence-based content |

### Advanced Upload Options

<AccordionGroup>
  <Accordion title="Bulk Upload">
    **Multiple File Selection**
    1. Click **New** → **New Video**
    2. Select multiple files (Ctrl/Cmd + click)
    3. Set common details for all videos
    4. Customize individual video details
    5. Publish all at once

    **Folder Upload**
    1. Drag entire folder to organization
    2. Nuclom processes all video files
    3. Automatically creates channels based on folder structure
  </Accordion>
  <Accordion title="Processing Quality">
    - **Standard** - Good quality, faster processing
    - **High** - Better quality, longer processing time
    - **Original** - Maintain original quality, longest processing
  </Accordion>
  <Accordion title="Privacy Settings">
    - **Organization** - Visible to all organization members
    - **Channel** - Visible to channel members only
    - **Private** - Only visible to you
    - **Custom** - Set specific permissions
  </Accordion>
</AccordionGroup>

## Performance Optimization

### File Preparation

<Note>
  Optimizing videos before upload improves processing time and playback quality.
</Note>

**Before Upload:**

- **Compress videos** - Balance quality and file size
- **Optimize resolution** - Don't exceed necessary quality
- **Trim unnecessary content** - Remove dead time
- **Use consistent format** - MP4 for best compatibility

**Recommended Tools:**

| Tool | Use Case |
| ---- | -------- |
| **HandBrake** | Free video compression |
| **Adobe Media Encoder** | Professional compression |
| **FFmpeg** | Command-line processing |

## Best Practices

### Naming Conventions

<Tabs>
  <Tab title="Descriptive Format">
    ```
    [Project] - [Topic] - [Date]
    Example: "Alpha Launch - User Testing Results - 2025-01-15"
    ```
  </Tab>
  <Tab title="Hierarchical Format">
    ```
    [Department] > [Project] > [Topic]
    Example: "Marketing > Q1 Campaign > Brand Guidelines"
    ```
  </Tab>
  <Tab title="Sequential Format">
    ```
    [Series Name] - [Episode Number] - [Topic]
    Example: "Onboarding Series - 01 - Welcome and Setup"
    ```
  </Tab>
</Tabs>

### Content Standards

**Technical Standards:**

- Minimum resolution: 720p (HD)
- Clear audio with minimal background noise
- Well-lit, professional appearance
- Proper composition and focus

**Content Standards:**

- Clear objectives defined
- Logical flow and organization
- Concise delivery
- Professional presentation

## Troubleshooting

<AccordionGroup>
  <Accordion title="Upload Failures">
    - Check file format compatibility
    - Verify internet connection stability
    - Ensure sufficient storage space
    - Try uploading smaller files first
  </Accordion>
  <Accordion title="Processing Delays">
    - Large files take longer to process
    - High-quality settings increase processing time
    - Server load can affect processing speed
  </Accordion>
  <Accordion title="Missing Videos">
    - Check organization and channel filters
    - Verify you're in the correct organization
    - Review permission settings
    - Search using different keywords
  </Accordion>
</AccordionGroup>
