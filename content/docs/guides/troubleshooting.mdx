## Quick fixes

Try these first:

1. Refresh the page (`Cmd/Ctrl + R`)
2. Log out and sign back in
3. Clear browser cache
4. Try Chrome or Edge

## Video playback

**Video won't play**
- Check your internet connection (5+ Mbps for HD)
- Disable ad blockers or privacy extensions
- Try a different browser

**No audio**
- Check device and player volume
- Try headphones to isolate the issue

## Uploads

**Upload failed**
- Supported formats: MP4, MOV, AVI, MKV
- Check file size against your plan limits
- Keep the browser tab open during upload

**Import not working**
- Reconnect the integration in **Organization Settings → Integrations**
- Verify you have access to the file in the source app

## AI processing

**Transcript or summary missing**
- Processing can take a few minutes for longer videos
- Check the status banner on the video page
- If stuck, contact support with the video URL

## Sharing

**Share link not working**
- Link may have expired — create a new one
- Password may be required — check with the sender
- View limit may be reached — generate a fresh link

**Access denied**
- Confirm you're signed into the correct organization
- Ask the video owner to update sharing permissions

## System requirements

| Requirement | Minimum |
| ----------- | ------- |
| Browser | Chrome, Edge, or Firefox (latest) |
| Internet | 5 Mbps |
| RAM | 4 GB |
