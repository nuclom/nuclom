## Quick Fixes

<Note>
  Try these first before diving deeper.
</Note>

1. Refresh the page (`Cmd/Ctrl + R`)
2. Log out and sign back in
3. Clear your browser cache
4. Try another browser (Chrome or Edge recommended)
5. Check your internet connection

## Video Playback

### Video Won’t Play

<Steps>
  <Step title="Refresh the page">
    Reload the video and try again.
  </Step>
  <Step title="Check your connection">
    Streaming needs a stable connection (5 Mbps+ for HD).
  </Step>
  <Step title="Disable extensions">
    Ad blockers and privacy tools can interfere with playback.
  </Step>
</Steps>

### Audio Missing

- Confirm your device volume is up.
- Check the player volume controls.
- Try another browser to isolate device issues.

## Uploads & Imports

### Upload Failed

| Check | Recommendation |
| ----- | -------------- |
| File format | MP4, MOV, AVI, MKV |
| File size | Stay within your plan limits |
| Connection | Use a stable connection and keep the tab open |

### Import Not Showing

<Steps>
  <Step title="Recheck integrations">
    Go to **Organization Settings** → **Integrations** and confirm the connection.
  </Step>
  <Step title="Try again">
    Reopen **Import Videos** and select the recording again.
  </Step>
</Steps>

## AI Processing

### Transcript or Summary Missing

<Steps>
  <Step title="Give it time">
    Processing may take a few minutes for longer videos.
  </Step>
  <Step title="Check the status banner">
    The video page shows processing status and current stage.
  </Step>
  <Step title="Reprocess if needed">
    If processing is stuck, contact support with the video URL.
  </Step>
</Steps>

## Sharing & Access

### Share Link Not Working

| Possible Cause | Fix |
| -------------- | --- |
| Link expired | Create a new share link |
| Password required | Enter the correct password |
| Max views reached | Generate a fresh link |

### Access Denied

- Confirm you’re signed into the correct organization.
- Ask the owner to update your role or sharing access.

## Integrations

### Integration Errors

- Reconnect the integration in **Organization Settings** → **Integrations**.
- Confirm you have permission in the external tool (Zoom, Google Meet, Slack, Teams).
- Try again after reconnecting.

## System Requirements

| Requirement | Recommendation |
| ----------- | -------------- |
| Browser | Latest Chrome, Edge, or Firefox |
| Internet | 5 Mbps+ for HD playback |
| Memory | 4 GB RAM minimum, 8 GB recommended |
