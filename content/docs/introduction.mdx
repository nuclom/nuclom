## Key Features

<AccordionGroup>
  <Accordion title="Multi-organization Support">
    Separate workspaces for different teams with role-based access control
  </Accordion>
  <Accordion title="AI-powered Insights">
    Automatic summaries and action item extraction from your videos
  </Accordion>
  <Accordion title="Time-stamped Comments">
    Leave feedback at specific video moments for precise collaboration
  </Accordion>
  <Accordion title="Smart Organization">
    Channels, series, and tags for structured content management
  </Accordion>
</AccordionGroup>

## Tech Stack

| Layer          | Technologies                                      |
| -------------- | ------------------------------------------------- |
| **Frontend**   | Next.js 16, React 19, TypeScript, Tailwind CSS    |
| **UI**         | shadcn/ui, Radix UI, Lucide Icons                 |
| **Backend**    | Next.js API Routes, Effect-TS                     |
| **Database**   | PostgreSQL, Drizzle ORM                           |
| **Auth**       | Better-Auth (GitHub, Google OAuth)                |
| **Storage**    | Cloudflare R2                                     |
| **AI**         | OpenAI, Vercel AI SDK                             |
| **Deployment** | Vercel with CI/CD                                 |
