## Version Format

This project follows [Semantic Versioning](https://semver.org/):

| Version Type | When to Use |
|--------------|-------------|
| **MAJOR** | Incompatible API changes |
| **MINOR** | Backwards compatible functionality additions |
| **PATCH** | Backwards compatible bug fixes |

## Deprecation Policy

<Note>
  When features are deprecated, they will continue to work for at least **6 months** before removal. Deprecated features will be clearly marked in the documentation with migration guides.
</Note>

## Staying Updated

To stay informed about API changes:

1. Watch this changelog for updates
2. Subscribe to our API announcements mailing list
3. Check the `X-API-Version` response header for version information
4. Review migration guides when upgrading
