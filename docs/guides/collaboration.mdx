| Level | Capabilities |
| ----- | ------------ |
| **Viewer** | Watch videos, view AI insights, basic sharing |
| **Contributor** | All viewer permissions plus upload and edit |

## Best Practices

### Creating Collaborative Content

**Structure for Clarity:**

- Clear introduction and objectives
- Logical flow and organization
- Key decisions clearly stated
- Summary and next steps

**Enable Discovery:**

- Use descriptive titles
- Add comprehensive descriptions
- Organize into appropriate channels
- Tag with relevant topics

### Managing Collaboration

<Warning>
  Set clear expectations for communication guidelines, response times, and decision documentation standards.
</Warning>

**Collaboration Roles:**

- Define content ownership
- Establish review responsibilities
- Set approval requirements
- Clarify decision-making authority

## Troubleshooting

<AccordionGroup>
  <Accordion title="Low Engagement">
    - Make content more discoverable
    - Use notifications strategically
    - Provide clear calls-to-action
    - Leverage AI insights
  </Accordion>
  <Accordion title="Information Overload">
    - Organize content into channels
    - Use AI summaries for quick review
    - Prioritize important notifications
    - Create focused content series
  </Accordion>
  <Accordion title="Sharing Link Issues">
    - Verify link permissions
    - Check expiration dates
    - Ensure recipients have appropriate access
    - Test links before sharing
  </Accordion>
</AccordionGroup>
