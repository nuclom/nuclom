Effective team management is crucial for successful video collaboration. This guide covers inviting members, managing roles and permissions, and maintaining organization security.

## Understanding Team Structure

### Organization Hierarchy

```
Organization Level
├── Team Level (sub-groups, max 20 per org)
├── Channel Level (topic-specific access)
└── Content Level (individual video permissions)
```

## Team Roles

### Organization Roles

| Role | Capabilities |
| ---- | ------------ |
| **Owner** | Full administrative control, billing, member management, only one per org |
| **Admin** | Member management (except owner), content moderation, settings |
| **Editor** | Content creation and editing, read access to channels and teams |
| **Member** | Upload, organize, comment, create channels/series (default role) |
| **Viewer** | View-only access, limited sharing, cannot upload |

### Content-Specific Roles

| Role | Scope |
| ---- | ----- |
| **Channel Admin** | Manage specific channel settings and membership |
| **Series Moderator** | Manage series content, order, and access |

## Inviting Team Members

### Individual Invitations

<Tabs>
  <Tab title="Via Email">
    <Steps>
      <Step title="Open Members">
        Go to **Settings** → **Members**
      </Step>
      <Step title="Click Invite">
        Click **Invite Member**
      </Step>
      <Step title="Enter Details">
        Enter email, select role, add optional message
      </Step>
      <Step title="Send">
        Click **Send Invitation**
      </Step>
    </Steps>
  </Tab>
  <Tab title="Via Sharing Link">
    1. Generate organization invitation link
    2. Set expiration date and usage limits
    3. Share link with team members
    4. Monitor invitation acceptance
  </Tab>
  <Tab title="Via Command Bar">
    1. Press **Cmd/Ctrl + K**
    2. Type "invite member"
    3. Enter email and role
    4. Send invitation
  </Tab>
</Tabs>

### Bulk Invitations

<AccordionGroup>
  <Accordion title="CSV Import">
    1. Prepare CSV file with columns: email, role, name
    2. Go to **Settings** → **Members** → **Bulk Import**
    3. Upload CSV file
    4. Review and send all invitations
  </Accordion>
  <Accordion title="Domain-Based Invitations">
    1. Set up domain-based auto-acceptance
    2. Anyone with company email can join
    3. Set default role for domain users
    4. Configure approval workflows if needed
  </Accordion>
</AccordionGroup>

### Invitation Status

| Status | Description |
| ------ | ----------- |
| **Pending** | Invitation sent, not yet accepted |
| **Accepted** | Member has joined organization |
| **Expired** | Invitation has expired |
| **Rejected** | Invitation was declined |

  </Step>
  <Step title="Search">
    Use search and filters to find specific members
  </Step>
</Steps>

### Changing Member Roles

**Individual Changes:**

1. Find the member in **Settings** → **Members**
2. Click **Edit** or the role dropdown
3. Select new role
4. Confirm changes

**Bulk Changes:**

1. Select multiple members
2. Choose **Change Role** from actions menu
3. Apply to all selected

## Handling Team Changes

### Member Departures

<Steps>
  <Step title="Revoke Access">
    Remove organization access immediately
  </Step>
  <Step title="Transfer Content">
    Transfer content ownership to appropriate team members
  </Step>
  <Step title="Cleanup">
    Remove from channels, revoke sharing permissions, archive info
  </Step>
</Steps>

### Team Restructuring

**Reorganization Planning:**

1. Identify affected members
2. Map current permissions
3. Plan new structure
4. Communicate changes clearly

## Troubleshooting

<AccordionGroup>
  <Accordion title="Member Can't Access Organization">

  </Accordion>

    - Update access controls
  </Accordion>
