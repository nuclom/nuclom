# Video Upload and Organization

Master the art of video organization in Nuclom. This guide covers everything from uploading your first video to advanced organization strategies that will keep your content discoverable and well-structured.

## What You'll Learn

- Video upload process and best practices
- Organizing videos with channels and series
- Video metadata and search optimization
- Advanced organization strategies
- Bulk operations and automation
- Performance optimization tips

## Video Upload Process

### Supported Formats

**Recommended Formats**

- **MP4** - Best compatibility and performance
- **MOV** - Good for Mac users
- **AVI** - Universal compatibility
- **MKV** - High quality with good compression

**Video Specifications**

- **Resolution** - Up to 4K (3840x2160)
- **Frame Rate** - 24fps, 30fps, 60fps
- **Bitrate** - Automatic optimization
- **Audio** - AAC, MP3, WAV supported

### Step-by-Step Upload

#### Method 1: Via New Button

1. Click **"New"** in the top navigation
2. Select **"New Video"**
3. Choose your video file
4. Fill in video details while uploading
5. Click **"Publish"** when ready

#### Method 2: Via Drag and Drop

1. Drag video files directly to the organization
2. Drop them in the main content area
3. Video details form will appear
4. Complete the information and publish

#### Method 3: Via Command Bar

1. Press **⌘K** (Mac) or **Ctrl+K** (Windows/Linux)
2. Type "upload video"
3. Select the upload option
4. Choose your file and continue

### Video Details and Metadata

#### Required Information

**Title**

- Make it descriptive and searchable
- Include key topics or themes
- Avoid generic titles like "Video 1"
- Example: "Q4 Product Demo - New Dashboard Features"

**Duration**

- Automatically detected during upload
- Helps viewers plan their time
- Used in search and filtering

#### Optional but Recommended

**Description**

- Detailed explanation of video content
- Include key points and timestamps
- Add relevant keywords for searchability
- Use markdown formatting for better structure

**Thumbnail**

- Nuclom generates options automatically
- Choose the most representative frame
- Upload custom thumbnail if needed
- Recommended size: 1280x720 pixels

**Channel Assignment**

- Organize videos by topic or project
- Create new channels during upload
- Helps with content discovery
- Can be changed later

**Series Assignment**

- Add to existing series or create new one
- Maintains logical viewing order
- Perfect for tutorials or ongoing content
- Supports automatic progression

### Advanced Upload Options

#### Bulk Upload

**Multiple File Selection**

1. Click **"New"** → **"New Video"**
2. Select multiple files (Ctrl/Cmd + click)
3. Set common details for all videos
4. Customize individual video details
5. Publish all at once

**Folder Upload**

1. Drag entire folder to organization
2. Nuclom will process all video files
3. Automatically create channels based on folder structure
4. Maintain file organization

#### Upload Settings

**Processing Quality**

- **Standard** - Good quality, faster processing
- **High** - Better quality, longer processing time
- **Original** - Maintain original quality, longest processing

**Transcription Options**

- **Automatic** - AI-generated transcripts
- **Manual** - Upload your own transcript
- **None** - Skip transcription

**Privacy Settings**

- **Organization** - Visible to all organization members
- **Channel** - Visible to channel members only
- **Private** - Only visible to you
- **Custom** - Set specific permissions

**During Video Upload**

1. In the channel dropdown, select "Create New Channel"
2. Enter channel name and description
3. Video will be automatically added to the new channel

**From Channels Page**

1. Navigate to **Channels** in your organization
2. Click **"Create Channel"**
3. Fill in channel details
4. Set permissions and settings

5. Set series order and progression settings

**From Series Page**

1. Navigate to **Series** in organization
2. Click **"Create Series"**
3. Fill in series details
4. Add videos to the series

## Performance Optimization

### Video Optimization

#### File Preparation

**Before Upload**

- **Compress videos** - Balance quality and file size
- **Optimize resolution** - Don't exceed necessary quality
- **Trim unnecessary content** - Remove dead time or irrelevant sections
- **Consistent format** - Use MP4 for best compatibility

**Recommended Tools**

- **HandBrake** - Free video compression
- **Adobe Media Encoder** - Professional compression
- **FFmpeg** - Command-line video processing
- **Online converters** - Quick web-based solutions

#### Upload Optimization

**Network Considerations**

- **Upload during off-peak hours** - Faster processing
- **Use wired connection** - More stable than WiFi
- **Close unnecessary applications** - Free up bandwidth
- **Monitor upload progress** - Ensure successful completion

**Batch Processing**

- **Upload smaller batches** - Reduce chance of failures
- **Stagger uploads** - Avoid overwhelming the system
- **Use queue management** - Organize upload order

## Troubleshooting

### Common Upload Issues

**Upload Failures**

- Check file format compatibility
- Verify internet connection stability
- Ensure sufficient storage space
- Try uploading smaller files first

**Processing Delays**

- Large files take longer to process
- High-quality settings increase processing time
- Server load can affect processing speed
- Processing times vary by file complexity

**Quality Issues**

- Check source video quality
- Verify upload settings
- Consider re-uploading with different settings
- Contact support for persistent issues

### Organization Problems

**Missing Videos**

- Check organization and channel filters
- Verify you're in the correct organization
- Review permission settings
- Search using different keywords

**Incorrect Channel Assignment**

- Edit video settings to change channel
- Use bulk operations for multiple videos
- Review auto-assignment rules
- Check channel permissions

**Series Order Issues**

- Verify series sorting settings
- Use manual ordering if needed
- Check for duplicate videos in series
- Review series progression settings
